# Zalo Mini App Skills

Thư mục này chứa các tài liệu skills dành cho AI Agent (Cursor, Claude) để hỗ trợ phát triển Zalo Mini App.

## Cấu trúc thư mục

```
skills/
├── README.md              # File này - Hướng dẫn sử dụng skills
├── general.md             # Tổng quan về Zalo Mini App
├── zmp-sdk-api.md         # API Reference cho ZMP SDK
├── zaui-react.md          # ZaUI React Components (v1.10.x)
├── authentication.md      # Hướng dẫn xác thực và đăng nhập
├── payment.md             # Tích hợp thanh toán (Checkout SDK)
├── app-config.md          # Cấu hình ứng dụng (app-config.json)
├── permissions.md         # Hướng dẫn xin cấp quyền
├── devtools.md            # Công cụ phát triển (CLI, Extension)
├── deployment.md          # Triển khai và phát hành Mini App
├── troubleshooting.md     # Xử lý lỗi thường gặp
└── best-practices.md      # Best practices khi phát triển
```

## Cách sử dụng

### Với Cursor
Cursor sẽ tự động đọc các file trong thư mục này để hiểu context về dự án Zalo Mini App và cung cấp gợi ý code chính xác hơn.

### Với Claude (hoặc các AI khác)
Có thể cung cấp nội dung các file này như context để AI hiểu về Zalo Mini App platform.
